
# 매장/전시회 안내 로봇(목표물 탐색)
> 웹으로 관람하고자 하는 작품 목록을 선택하면, 그 목록과 지도 기반으로 관람 동선을 계획하고 해당 작품 앞에 도착 시 해설해주는 로봇
1. AD
  - 저장된 미술관 지도 기반으로 자율주행
  - rgb 카메라를 통한(테슬라식 자율 주행) 장애물 극복
    - 저장된 지도 만으로는 판단할 수 없는 "다른 관람객"등의 장애물을 가정
  - 작품 목록 갱신 시, 실시간으로 자율주행 동선 수정
2. AI
  - 그림을 객체인식하여 `미정` 기능 구현
    - 그림은 "유일"하기 때문에 학습 + 객체인식에 유리할 것이라 생각
  - 자율 주행 시 예기치 못한 장애물 극복에 객체인식 활용
  - 관람 중 관심이 생기는 작품이 있다면, 음성인식 AI를 통한 작품 목록 실시간 갱신
3. HW
  - 차량(로봇) 하드웨어 모델링 및 제작(3D프린팅, 선반가공, 레이저가공 등 활용)
  - PCB 설계 및 최종본 확정 시 인쇄
4. Web
  - 웹 접속, 음성인식 AI 등으로 작품 목록을 갱신하면, 자율주행 동선을 실시간으로 변경

- 차별화: 시각 목표 발견 -> 즉석 경로 생성
  QR/마커 없이 로고/포스터/특정 사물만으로 목표 인식 후 즉시 이동
- 추가 가치: 부스 배치 변경에도 표지판만 있으면 작동
- MVP: 로고 데이터셋 소량 학습 -> 찾기 성공률 측정

2) 창고 피킹 보조 카트
- 차별화: 라벨 없는 박스도 식별
  바코드 대신 박스 모서리 패턴/테이프/텍스처로 유사 박스 분류 수행
- 추가 가치: 라벨 훼손/오염 상황에서도 작업 연속성 확보
- MVP: 박스 외형 특징 기반 분류 + 목표 박스 접근
